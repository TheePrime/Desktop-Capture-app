DESKTOP CAPTURE APP - QUICK SETUP GUIDE
========================================

INSTALLATION (5 minutes)
------------------------

STEP 1: Install Desktop App
   1. Double-click "Desktop Capture Setup 1.0.0.exe"
   2. Click "Install" in the wizard
   3. Wait for installation to complete
   4. Click "Finish"

STEP 2: Install Chrome Extension
   1. Unzip "desktop-capture-extension.zip" to any folder
   2. Open Chrome browser
   3. Type in address bar: chrome://extensions
   4. Turn ON "Developer mode" (toggle in top-right corner)
   5. Click "Load unpacked" button
   6. Select the unzipped "extension" folder
   7. You should see "Desktop Capture Extension" appear

STEP 3: Start Capturing
   1. Launch "Desktop Capture" from your desktop shortcut
   2. Click the "Start Tracking" button
   3. Status should turn green and say "Active"
   4. Open Chrome and go to any website (Twitter, LinkedIn, etc.)
   5. Click on posts, images, or any content
   6. Go back to the app's "Activities" tab to see captured data!

WHERE IS MY DATA?
-----------------
All screenshots and click data are automatically saved to:

   C:\Users\[YourUsername]\Documents\DesktopCapture\

You can change this location in the app Settings.

DAILY USE
---------
1. Open the Desktop Capture app
2. Click "Start Tracking"
3. Browse the web normally
4. Check the Activities tab to review your clicks
5. Click "Stop" when done
